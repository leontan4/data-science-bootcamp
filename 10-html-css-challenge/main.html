Summary Page

The purpose of this project is to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled the data from OpenWeatherMap API to assemble a dataset more than uniquely random 500 countries. Once the dataset was assembled, Matplolib was used to plot the various aspect of the weather vs latitude.
Other variables were included in our analysis as well:

  1. Temperature
  2. Humidity
  3. Cloudiness
  4. Wind Speed
This site includes the raw data used in our analysis in looking for correlation between variables. Comparison graph were also included to provide better comprehension and vizualization.
